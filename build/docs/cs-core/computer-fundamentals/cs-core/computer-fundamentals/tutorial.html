فهرست یادگیری مبانی کامپیوتر

این آموزش مبانی کامپیوتر از مباحث پایه تا مفاهیم پیشرفته را پوشش می‌دهد. در این دوره با بخش‌هایی مانند سخت‌افزار، نرم‌افزار، سیستم عامل، لوازم جانبی و دیگر اجزای کامپیوتر آشنا می‌شوید.

/ 5

چرا باید مبانی کامپیوتر یاد بگیریم؟

یادداشت

کامپیوتر شما می‌تواند پیچیده‌ترین مسائل را در چند میلی‌ثانیه حل کند!

یادگیری مبانی کامپیوتر باعث می‌شود:

  • بهتر بفهمید کامپیوتر چگونه کار می‌کند و چگونه مسائل را حل می‌کند.
  • یادگیری تکنولوژی‌های جدید برای شما آسان‌تر شود و فرصت‌های شغلی بیشتری ایجاد شود.
  • توانایی استفادهٔ ایمن و کارآمد از کامپیوتر و نرم‌افزارها را پیدا کنید.
  • بتوانید با سرعت تغییرات دنیای فناوری هماهنگ شوید.
  • در مسیر شغلی خود موفق باشید، زیرا تقریباً همهٔ مشاغل امروزی از کامپیوتر استفاده می‌کنند.

1. مقدمه

کامپیوترها به‌طور کامل وارد زندگی ما شده‌اند. این دستگاه‌ها که زمانی فقط برای محاسبه طراحی شده بودند، امروز به سیستم‌هایی قدرتمند برای پردازش داده، ارتباطات و مدیریت اطلاعات تبدیل شده‌اند.

در این بخش با موارد زیر آشنا می‌شوید:

  • کامپیوتر و اجزای آن
  • کاربرد های کامپیوتر

2. اجزای کامپیوتر

یک کامپیوتر دارای سه بخش اصلی است: CPU، دستگاه‌های ورودی (Input Devices) و دستگاه‌های خروجی (Output Devices). وظیفهٔ CPU پردازش داده‌های ورودی و ارائهٔ نتیجه از طریق خروجی‌هاست.

در این بخش اجزای مختلف کامپیوتر را بررسی می‌کنیم:

  • سخت افزار (Hardware)
  • نرم افزار (Software)
  • واحد پردازش مرکزی (Central Processing Unit (CPU))
  • حافظه کامپیوتر (Computer Memory)
  • دستگاه های ورودی (Input Devices)
  • دستگاه های خروجی (Output Devices)

3. سیستم عامل ها (Operating Systems)

سیستم عامل (Operating System به اختصار OS) نرم‌افزار سیستمی است که منابع سخت‌افزاری و نرم‌افزاری کامپیوتر را مدیریت کرده و اجرای برنامه‌ها را ممکن می‌کند.

این بخش شامل:

  • معرفی
  • Windows
  • Linux
  • iOS
  • macOS
  • ChromeOS

4. اینترنت و شبکه جهانی وب (Internet & World Wide Web (WWW))

اینترنت یک شبکهٔ جهانی از میلیون‌ها سیستم و دستگاه متصل است. WWW یا World Wide Web یک سرویس مبتنی بر اینترنت است که شامل صفحات وب، محتوای آنلاین و سرویس‌های اینترنتی می‌شود.

در این بخش با موضوعات زیر آشنا می‌شوید:

  • مرورگر وب (Web Browser)
  • وب سرور (Web Server)
  • پروتکل های وب (Web Protocols)
  • امنیت وب (Web Security)
  • پایگاه های داده (Databases)
  • اینترنت (Internet)
  • WiFi

5. ابزار های Office

ابزارهای Office نرم‌افزارهایی هستند که برای کارهای اداری مانند نوشتن متن، ساخت ارائه، مدیریت داده و همکاری تیمی استفاده می‌شوند.

این بخش شامل آموزش ابزارهای زیر است:

  • پردازش متنی (Word Processing) : MS Word و Google Docs
  • صفحه گسترده (Spreadsheets) : MS Excel و Google Sheets
  • ارائه (Presentations) : MS PowerPoint و Google Slides
  • ایمیل (Email Clients) : Gmail و Outlook
  • ابزارهای همکاری (Collaboration Tools) : OneDrive، Google Drive و Dropbox

6. کلیدهای میانبر (Shortcut Keys)

کلیدهای میانبر (Shortcut Keys) ترکیب‌هایی از کلیدهای صفحه‌کلید هستند که باعث افزایش سرعت و بهبود کارایی می‌شوند.

در این بخش با میانبرهای مهم آشنا می‌شوید:

  • کلیدهای میانبر عمومی
  • کلیدهای عملگر صفحه‌کلید (Keyboard)
  • کلیدهای میانبر Windows
  • کلید های میانبر Ubuntu

7. مقایسه‌های مهم

موضوعات زیر برای هر دانشجوی علوم کامپیوتر ضروری هستند:

  • مقایسهٔ رم (RAM) و رام (ROM)
  • مقایسهٔ هارد دیسک (Hard Disk) و فلاپی دیسک (Floppy Disk)
  • مقایسهٔ سی‌دی‌ (CD-ROM) و دیسک‌های مغناطیسی (Magnetic Disks)
  • مقایسهٔ تشخیص نوری کاراکتر (OCR) و MICR
  • مقایسهٔ دیسک مغناطیسی (Magnetic Disk) و دیسک نوری (Optical Disk)
  • مقایسهٔ حافظه های HDD و SSD
  • مقایسهٔ سی‌دی (CD) و دی‌وی‌دی (DVD)
  • مقایسهٔ بلوری (Blu-Ray) و دی‌وی‌دی (DVD)
  • مقایسهٔ نرم‌افزار (Software) و نرم‌افزار کاربردی (Utility Software)
  • مقایسهٔ نرم‌افزار (Software) و سیستم‌عامل (Operating System)
  • مقایسهٔ نرم‌افزار سیستمی (System Software) و نرم‌افزار کاربردی (Application Software)
  • مقایسهٔ بارکد (Barcode) و کد QR

9. شبکه‌های کامپیوتری (Computer Networks)

در این بخش با مفاهیم پایه شبکه آشنا می‌شوید:
